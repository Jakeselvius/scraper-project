# scraper-project
This is a project I built to help better my understandig of web scaraping 
and anaylizing that data

This program: 
- Scrapes 100+ books across multiple pages (title, price, star rating)

- Cleans and stores data in a SQLite database

- Runs SQL queries to find patterns (cheapest 5-star books, averages by rating, etc.)

- Generates data visualizations using Pandas and Matplotlib

- Organized into clean, modular Python files

Feel free to take a peek. Any feedback is welcomed, I'm always trying to get better.
